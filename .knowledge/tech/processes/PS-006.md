## 背景

ERCOT（Electric Reliability Council of Texas）管理德州电网，覆盖 2600 万用户、90% 负荷。ERCOT 电力市场以极端波动著称（2021 寒潮电价飙至 $9000/MWh 上限），是研究极端天气-电力联动的理想对象。

本条目基于 2026-07-22~23 的实际下载测试验证，通过 **EIA API v2 + GridStatus.io API** 双源互补，成功获取 ERCOT 2025-01-01 至今的完整电力市场数据（电价 + 负荷 + 发电燃料类型），共 40 个文件 / 42.6 MB / 344,460 行。

## 第一部分：EIA API v2 — 负荷与发电数据

### 注册与认证

1. 访问 https://www.eia.gov/opendata/register.php
2. 填写邮箱注册，key 立即发送到邮箱
3. 设置环境变量：`$env:EIA_API_KEY = "你的key"`

### 数据路由

| 路由 | 数据内容 | 频率 | ERCOT respondent |
|------|---------|------|-----------------|
| `electricity/rto/region-data/data/` | 负荷(D)/日前预测(DF)/净发电(NG)/交换(TI) | 小时级 | `ERCO` |
| `electricity/rto/fuel-type-data/data/` | 太阳能/风电/煤电/天然气/核电/储能/水电/其他 | 小时级 | `ERCO` |

### 关键参数

```python
# EIA API v2 请求参数
params = {
    "api_key": api_key,
    "frequency": "hourly",
    "data[0]": "value",
    "facets[respondent][]": "ERCO",  # ERCOT 的 respondent 代码
    "start": "2025-01-01T00",
    "end": "2025-02-01T00",
    "offset": 0,
    "length": 5000,  # 单次最大返回
    "sort[0][column]": "period",
    "sort[0][direction]": "asc",
}
```

### 陷阱提醒

- **respondent 代码是 `ERCO`**，不是 `ER_CO`（EIA 文档中带下划线的是 region-data 的旧格式）
- **`total` 字段返回字符串类型**，需要 `int()` 转换后才能用于分页比较
- **`sort` 参数用扁平格式** `sort[0][column]`，不要用 JSON 数组格式（requests 库会错误序列化）

## 第二部分：GridStatus.io API — 电价数据

### 注册与认证

1. 访问 https://www.gridstatus.io/sign-up 注册免费账号
2. 登录后访问 https://www.gridstatus.io/settings/api 获取 API key
3. 安装客户端：`pip install gridstatusio`
4. 设置环境变量：`$env:GRIDSTATUS_API_KEY = "你的key"`

### 免费方案额度

| 限制项 | 额度 |
|--------|------|
| 每月 API 请求 | 250 次 |
| 每月返回行数 | 500,000 行 |
| 单次响应行数 | 50,000 行 |
| 每秒请求 | 1 次 |
| 每分钟请求 | 30 次 |

### 数据集

| 数据集 ID | 名称 | 频率 | 最早数据 |
|-----------|------|------|---------|
| `ercot_spp_day_ahead_hourly` | 日前市场 SPP | 小时级 | 2010-12 |
| `ercot_spp_real_time_15_min` | 实时市场 SPP | 15分钟级 | 2010-12 |

### 交易枢纽

| 枢纽代码 | 名称 | 状态 |
|---------|------|------|
| HB_NORTH | 北部枢纽 | 可用 |
| HB_SOUTH | 南部枢纽 | 可用 |
| HB_HOUSTON | 休斯顿枢纽 | 可用 |
| HB_WEST | 西部枢纽 | 可用 |
| HB_PANHANDLE | 锅柄枢纽 | 无数据（已弃用） |

### 下载代码

```python
import gridstatusio

client = gridstatusio.GridStatusClient(api_key="你的key")

# 下载 DAM HB_NORTH 全部历史
df = client.get_dataset(
    "ercot_spp_day_ahead_hourly",
    start="2025-01-01",
    end="2026-07-23",
    filter_column="location",
    filter_value="HB_NORTH",
    verbose=False,
)
df.to_csv("ercot_dam_HB_NORTH.csv", index=False)
```

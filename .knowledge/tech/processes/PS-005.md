## 背景

SURFRAD（Surface Radiation Budget Network）是 NOAA 全球监测实验室运营的美国地表辐射收支观测网，提供**仪器实测**的太阳辐照数据（GHI/DNI/DHI）、长波辐射、紫外、光合有效辐射（PAR）及基本气象要素，时间分辨率 **1 分钟**。与卫星衍生产品（NSRDB/PVGIS）不同，SURFRAD 是验证卫星反演和数值模式的基准实测网络。

本条目基于 2026-07-20 的实际下载测试验证，**匿名 HTTPS 访问完全成功，已下载 7 站点 × 7 天 = 63 个文件**。

### 目录结构

```
https://gml.noaa.gov/aftp/data/radiation/surfrad/
├── {station}/                    # 历史目录（质控后归档）
│   └── {YYYY}/
│       └── {station}{YY}{DOY}.dat    # 每天一个文件
└── realtime/                     # 实时目录（当天数据实时追加）
    └── {station}/
        └── {station}{YY}{DOY}.dat
```

**文件命名**：`{station}{YY}{DOY}.dat`
- `bon26001.dat` = Bondville, 2026 年, DOY 001 = 2026-01-01
- `bon26201.dat` = Bondville, 2026 年, DOY 201 = 2026-07-20

### 数据格式（48 列）

**列顺序**：`year, jday, month, day, hour, minute, dec_hour, sza, (value, qc) × 20`

> ⚠️ **注意**：列顺序是 `year jday month day`，不是 `year month day jday`。jday（儒略日）在第 2 列。

| 列号 | 字段 | 含义 | 单位 |
|------|------|------|------|
| 0 | year | 年 | — |
| 1 | jday | 儒略日（DOY） | — |
| 2 | month | 月 | — |
| 3 | day | 日 | — |
| 4 | hour | 时（UTC） | — |
| 5 | minute | 分 | — |
| 6 | dec_hour | 小数小时 | — |
| 7 | sza | 太阳天顶角 | 度 |
| 8-9 | ghi / ghi_qc | **全球水平辐照度** + QC | W/m² |
| 10-11 | up_solar / qc | 上行短波（反射） | W/m² |
| 12-13 | dni / dni_qc | **直接法向辐照度** + QC | W/m² |
| 14-15 | dhi / dhi_qc | **散射水平辐照度** + QC | W/m² |
| 16-19 | dw_ir* | 下行长波 + case/dome 温度 | W/m², K |
| 20-23 | uw_ir* | 上行长波 + case/dome 温度 | W/m², K |
| 24-25 | uvb / qc | 紫外 B 辐照度 | mW/m² |
| 26-27 | par / qc | 光合有效辐射 | W/m² |
| 28-29 | net_solar / qc | 净短波 | W/m² |
| 30-31 | net_ir / qc | 净长波 | W/m² |
| 32-33 | net_total / qc | 净总辐射 | W/m² |
| 34-35 | temp / qc | 气温 | °C |
| 36-37 | rh / qc | 相对湿度 | % |
| 38-39 | windspd / qc | 风速 | m/s |
| 40-41 | winddir / qc | 风向 | 度 |
| 42-43 | pressure / qc | 气压 | hPa |

**缺失值**：`-9999.9`

**QC 码**：0=未检查，1=坏，2=可疑，3=好

### Python 下载示例

```python
import urllib.request
import os

BASE_URL = "https://gml.noaa.gov/aftp/data/radiation/surfrad/"
STATIONS = ['bon','dra','fpk','gwn','psu','sxf','tbl']

def download_file(station, year, doy, out_dir="."):
    """下载单个 DOY 文件"""
    fname = f"{station}{str(year)[-2:]}{doy:03d}.dat"
    # 先试历史目录
    url = f"{BASE_URL}{station}/{year}/{fname}"
    try:
        urllib.request.urlretrieve(url, os.path.join(out_dir, fname))
        return fname, 'historical'
    except:
        pass
    # 404 则试 realtime
    url = f"{BASE_URL}realtime/{station}/{fname}"
    try:
        urllib.request.urlretrieve(url, os.path.join(out_dir, fname))
        return fname, 'realtime'
    except:
        return None, 'failed'

# 下载 Bondville 2026 年 DOY 200（7/19）
fname, status = download_file('bon', 2026, 200)
print(f"下载: {fname} ({status})")
```

### Python 解析示例

```python
import pandas as pd

COLUMNS = ['year','jday','month','day','hour','minute','dec_hour','sza',
           'ghi','ghi_qc','up_solar','up_solar_qc','dni','dni_qc','dhi','dhi_qc',
           'dw_ir','dw_ir_qc','dw_ir_case','dw_ir_case_qc','dw_ir_dome','dw_ir_dome_qc',
           'uw_ir','uw_ir_qc','uw_ir_case','uw_ir_case_qc','uw_ir_dome','uw_ir_dome_qc',
           'uvb','uvb_qc','par','par_qc','net_solar','net_solar_qc',
           'net_ir','net_ir_qc','net_total','net_total_qc',
           'temp','temp_qc','rh','rh_qc','windspd','windspd_qc',
           'winddir','winddir_qc','pressure','pressure_qc']

def parse_surfrad(filepath):
    """解析 SURFRAD .dat 文件"""
    with open(filepath, 'r', errors='ignore') as f:
        lines = f.readlines()
    records = []
    for line in lines[2:]:  # 跳过前 2 行头部
        tokens = line.split()
        if len(tokens) == 48:
            records.append([float(t) for t in tokens])
    df = pd.DataFrame(records, columns=COLUMNS)
    # 构造 datetime
    dt_str = (df['year'].astype(int).astype(str) + '-' +
              df['month'].astype(int).astype(str).str.zfill(2) + '-' +
              df['day'].astype(int).astype(str).str.zfill(2) + ' ' +
              df['hour'].astype(int).astype(str).str.zfill(2) + ':' +
              df['minute'].astype(int).astype(str).str.zfill(2))
    df['datetime'] = pd.to_datetime(dt_str, format='%Y-%m-%d %H:%M')
    # 缺失值处理
    for col in ['ghi','dni','dhi','temp','rh','windspd','pressure']:
        df[col] = df[col].where(df[col] > -1000)
    return df

df = parse_surfrad('bon26200.dat')
print(f"记录数: {len(df)}")
print(f"GHI: max={df['ghi'].max():.1f} W/m², DNI: max={df['dni'].max():.1f} W/m²")
```

### 与其他辐射数据源对比

| 数据源 | 类型 | 分辨率 | 覆盖 | 访问门槛 | 实时性 |
|--------|------|--------|------|---------|--------|
| **SURFRAD** | **实测** | **1 分钟** | 美国 7 站 | **匿名** | **当天实时** |
| BSRN | 实测 | 1 分钟 | 全球 ~80 站 | PANGAEA 注册 | 数月-1 年 |
| ARM | 实测 | 秒-分钟 | 全球（含移动） | 注册账户 | ~48 小时 |
| GEBA | 实测 | 月均 | 全球 ~2500 站 | ETH 注册 | 1-2 年 |
| NSRDB | 卫星衍生 | 30 分钟 | 美国 4km 网格 | 免费 | 滞后数月 |
| PVGIS | 卫星+再分析 | 小时 | 全球 | 免费 | 历史 |

**关键区别**：SURFRAD 是**仪器实测**，NSRDB/PVGIS 是**卫星反演**。做辐照实测验证应优先用 SURFRAD/BSRN。

## 适用场景

- 太阳能资源评估与电站出力验证
- 卫星辐照产品（NSRDB/PVGIS）精度验证
- 数值模式辐射参数化验证
- 辐射气候学研究
- 光伏/光热系统性能分析
- 紫外（UV-B）和光合有效辐射（PAR）研究

## 不适用场景

- 美国本土外区域（仅 7 站，应选 BSRN 或 GEBA）
- 实时业务预报（当天数据实时追加，但历史目录有 4 天质控延迟）
- 全球覆盖分析（应选 NSRDB 网格化产品或 ERA5 再分析）
- 长期历史趋势（1995 年起，更早的数据应选 GEBA）
